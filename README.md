## ✨ Features (Khasiyaat)

- **📝 Create, Read, Update, Delete (CRUD):** Asani se naye tasks banayein, edit karein ya delete karein.
- **💾 Local Storage:** Aapka data aapke browser mein save rehta hai. Page refresh karne par bhi tasks delete nahi hote.
- **📊 Progress Tracker:** Visual progress bar jo batata hai ke aapne kitne feesad (percent) tasks complete kar liye hain.
- **🚩 Priority Tags:** Tasks ko *High*, *Medium*, ya *Low* priority mein set karein.
- **📅 Due Dates:** Har task ke sath ek deadline date set karne ka option.
- **🔍 Smart Filters:** Apne tasks ko *All*, *Active*, aur *Completed* ke hisaab se filter karein.
- **📱 Fully Responsive:** Mobile, tablet aur desktop par bilkul perfect nazar aati hai.
